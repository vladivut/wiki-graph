# Wiki Graph

## About

This project is based on the original repository:
https://github.com/blinpete/wiki-graph

This is an updated fork of the original project with improvements to the UI, API integration, and overall behavior.

Try it [here](https://vladivut.github.io/wiki-graph/?lang=en&linkLimit=10&maxDepth=2&wordle=&query=The%20Rolling%20Stones).

Note on APIs:

- autocomplete uses [MediaWiki Action API](https://www.mediawiki.org/wiki/API:Main_page)
- edges are built using the [links property](https://www.mediawiki.org/wiki/API:Links) from the MediaWiki Action API
